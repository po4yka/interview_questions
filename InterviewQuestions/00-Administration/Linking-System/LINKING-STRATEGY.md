# Linking Strategy

**Status**: Production Ready
**Last Updated**: 2025-11-25

## Core Requirements
- Every Q&A note **MUST** link to ≥1 MOC in YAML `moc` field
- Every Q&A note **MUST** have ≥2 related links in YAML `related` field
- Links use format: `moc-algorithms`, `c-hash-map`, `q-two-sum--algorithms--easy`
