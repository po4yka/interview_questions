# Summary (EN)

C# (pronounced "C sharp") is a modern, object-oriented programming language developed by Microsoft for building applications on the .NET platform, including web, desktop, mobile, cloud, and game development. It combines a syntax familiar to C/C++/Java developers with strong type safety, garbage collection, and rich standard libraries. C# is widely used in enterprise systems, backend services (ASP.NET Core), and game development with Unity, making it a core language for many interview scenarios.

## Key Points (EN)

- Strongly typed, object-oriented language with support for classes, interfaces, generics, delegates, LINQ, async/await, and functional-style constructs.
- Runs primarily on the .NET runtime (including cross-platform .NET), offering a comprehensive Base Class Library (BCL) for I/O, networking, collections, and more.
- Memory management via garbage collection reduces common errors like manual deallocation issues, while features like nullability annotations help avoid null reference bugs.
- Commonly used with ASP.NET Core for web APIs and services, and with Unity for game development, making it relevant in both enterprise and real-time application contexts.
- Evolving language (C# 6/7/8/9+) with frequent additions (pattern matching, records, spans, async streams) that improve expressiveness, safety, and performance.

## Ключевые Моменты (RU)

- Строго типизированный объектно-ориентированный язык с поддержкой классов, интерфейсов, дженериков, делегатов, LINQ, async/await и функциональных конструкций.
- Работает поверх рантайма .NET (включая кроссплатформенный .NET), предоставляя обширную базовую библиотеку (BCL) для работы с вводом-выводом, сетью, коллекциями и др.
- Управление памятью через сборщик мусора снижает риск ошибок ручного освобождения, а аннотации nullability помогают избегать ошибок обращения к null.
- Широко используется с ASP.NET Core для веб-API и сервисов, а также с Unity для разработки игр, что делает его актуальным для корпоративных и высоконагруженных интерактивных решений.
- Язык активно развивается (C# 6/7/8/9+): добавляются pattern matching, records, spans, async streams и другие возможности для повышения выразительности, безопасности и производительности.
