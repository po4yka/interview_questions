## Key Points (EN)

- Null-safety and type system: Understanding nullable vs non-null types, safe calls, Elvis operator, smart casts, and how Kotlin reduces NPEs compared to Java.
- Functions and functional style: Confident use of higher-order functions, lambdas, scope functions (let, run, apply, also, with), and collection operations (map, filter, flatMap, etc.).
- Object-oriented and language features: Data classes, sealed classes, object/singleton declarations, interfaces with default implementations, and visibility/modifier nuances.
- Coroutines basics: Launching coroutines, suspend functions, structured concurrency, and when to use coroutines instead of callbacks/threads.
- Java interoperability and idiomatic Kotlin: Calling Java from Kotlin (and vice versa), using nullability annotations, and recognizing/replacing Java-style code with idiomatic Kotlin constructs.
